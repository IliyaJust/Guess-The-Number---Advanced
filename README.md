# 🎯 Guess The Number – GTN

A simple number-guessing game with a powerful plugin system, built for developers who love to tinker.

---

## ✨ Features

- 🎮 Classic number-guessing gameplay with adjustable difficulty
- 🧩 Plugin system (coming soon) – write extensions in Python
- 📊 Record tracking and achievement system
- 🔧 Hidden `DEV MODE` for advanced users
- 🛠️ Modular reset system (`RESET RECORDS_DATA`, `RESET ALL`, etc.)
- ❤️ Lightweight, runs locally – no server required

---

## 🚀 Getting Started

### For Windows Users (Executable version)

1. Download the latest `.exe` file from the [Releases](https://github.com/IliyaJust/Guess-The-Number---Advnced/releases) page.
2. Run the `.exe` file – no Python installation required!
3. Follow the on-screen menu to start playing.

If you want to run the game from source:
